# Zomato Restaurant Anaysis

This interactive Power BI dashboard analyzes restaurant sales, order volume, and cuisine performance using historical Zomato restaurant order data. This project demonstrates data modeling, dashboard design, KPI development, and business storytelling to uncover trends across restaurants and cities.

📊 Project Overview

This dashboard explores restaurant performance by answering questions such as:

Which cities generate the highest restaurant sales?
Which cuisines are the most profitable?
Which restaurant brands outperform others?
How do order volume, quantity sold, and total sales compare across the dataset?

The report is designed to allow users to switch between key performance metrics while drilling into sales by city, cuisine, and restaurant.

Additional Context

Key Insights
Delhi generated the highest overall restaurant sales.
North Indian and Indian cuisine categories consistently produced the highest revenue.
Domino's Pizza ranked as the highest-selling restaurant in the dataset.
Large metropolitan cities significantly outperform smaller markets in overall restaurant sales.
